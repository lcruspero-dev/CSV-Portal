pm2 start ecosystem.config.js --- Start all API in ecosystem config.
pm2 ls --- list all API running.
pm2 stop + instance id --- to stop the app
pm2 restart + instance id -- to run again after stop
pm2 logs +  instance id -- to see the logs
pm2 monit --- to monitor in Realtime the API


Persistent Restarts (After System Reboot)
To ensure your APIs restart automatically after a server reboot:

Save the PM2 process list: (save location  C:\Users\pc\.pm2\dump.pm2)
pm2 save

Enable the startup script:
pm2 startup
